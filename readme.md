# nPZero Configurator Installer

## Overview
This repository contains a zip which includes the installer for the nPZero Configurator, including selected older versions which can be downloaded if needed. 
The auto update feature in the application will look for new versions at launch, by referencing this repository. 

## Install procedure
To install the latest version of the nPZero Configurator complete the following steps: 
1) Download the zip file found in the */latest* folder of this repository.
2) Extract the content to a temporary folder
3) Run setup.exe
